# Message API

The API used for accepting messages through my portfolio.

## Local Setup

1. Create virtual environment:

```bash
$ python3 -m venv ./.venv
$ source .venv/bin/activate
$ (.venv) # virtual environment now activated
```

2. Installation:

```bash
$ (.venv) python3 -m pip install -r requirements.txt
```

2. Deactivating:

```bash
$ (.venv) deactivate
```
